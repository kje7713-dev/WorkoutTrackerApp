### 2. **Strength Training Tracking**

**Log every detail:**
- Sets, reps, weight
- RPE (Rate of Perceived Exertion) on a 1-10 scale
- RIR (Reps in Reserve) - how many more you could do
- Tempo (movement speed)
- Rest periods between sets
- Exercise notes and coaching cues

**Progressive Overload Made Easy:**
- Set "Delta Weight" to automatically add weight each week (e.g., +5 lbs)
- Set "Delta Sets" to add volume each week (e.g., +1 set)
- Watch your progression happen systematically

**Exercise Categories:**
- Squat variations
- Hinge movements (deadlifts, RDLs)
- Horizontal press (bench, push-ups)
- Vertical press (overhead, push press)
- Horizontal pull (rows)

## ðŸ¤” Common Questions

### **Q: Is my data stored in the cloud?**
A: No. All data is stored locally on your device. You control backups via export/import.

### **Q: Can I share programs with friends?**
A: Yes! Export a block as JSON and share the file. They can import it (Pro feature).

### **Q: What if I miss a workout?**
A: No problem. Navigate back to any week and complete it. You control the pace.

### **Q: Can I modify a block mid-cycle?**
A: Yes, but carefully. Changes affect future weeks. Consider cloning and modifying instead.

### **Q: How do I track supersets?**
A: Group exercises with Set Group IDs (advanced feature). They'll be displayed together.

### **Q: What if I don't know how to build a program?**
A: Start simple (3 days, 4-5 exercises). Or use AI Block Import to get programs from coaches or AI tools.

### **Q: Does the app suggest exercises?**
A: No. You choose exercises based on your goals, equipment, and preferences.

### **Q: Can I use this for CrossFit?**
A: Absolutely! Supports conditioning workouts, AMRAPs, EMOMs, and complex movements.

### **Q: How accurate is progression tracking?**
A: 100% accurate based on what you log. The app never estimatesâ€”it shows exactly what you recorded.
