## Color Palette

### Segment Type Colors
- **Orange** (#FF9500) - Warmup, Mobility
- **Blue** (#007AFF) - Technique
- **Purple** (#AF52DE) - Drill
- **Red** (#FF3B30) - Positional Spar, Rolling
- **Green** (#34C759) - Cooldown, Breathwork, Flow
- **Gray** (#8E8E93) - Lecture
- **Secondary** - Other

### Semantic Colors
- **Primary** - Main text (#000000 / #FFFFFF)
- **Secondary** - Supporting text (#8E8E93)
- **Accent** - Interactive elements (tint color)
- **Success** - Completed items (#34C759)
- **Warning** - Constraints, cautions (#FF9500)
- **Danger** - Safety, stop conditions (#FF3B30)

### Background Colors
- **System Background** - Card backgrounds
- **Secondary System Background** - Nested elements
- **Tertiary System Background** - Deep nested

## Typography Scale

1. **Title 2** (.title2) - Day name, main headers
2. **Headline** (.headline) - Card titles
3. **Subheadline** (.subheadline) - Section headers, body
4. **Body** (.body) - Standard text
5. **Caption** (.caption) - Meta information
6. **Caption 2** (.caption2) - Smallest text, badges

## Spacing System

- **4pt** - Minimal spacing (icon spacing)
- **6pt** - Tight spacing (chip internal)
- **8pt** - Default spacing (section elements)
- **12pt** - Medium spacing (cards, pills)
- **16pt** - Large spacing (margins, padding)
- **24pt** - Extra large (section breaks)

## Touch Target Sizes

- **Minimum**: 44pt Ã— 44pt (Apple HIG)
- **Pill Cards**: Auto-sized, minimum 44pt height
- **Card Headers**: Full width, minimum 60pt height
- **Chevron Buttons**: 44pt Ã— 44pt
- **Chip/Tag**: 32pt height (accessible with padding)

## Animation Timing

- **Fast**: 0.2s - Button presses, micro interactions
- **Medium**: 0.3s - Card expand/collapse, scrolling
- **Slow**: 0.5s - Page transitions (not used here)

All animations use `.easeInOut` curve for smooth, natural motion.
