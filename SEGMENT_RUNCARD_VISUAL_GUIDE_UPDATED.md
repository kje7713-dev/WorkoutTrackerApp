## Design Principles Applied

1. **Progressive Disclosure**
   - Only show sections with data
   - No empty sections cluttering the UI
   - Graceful degradation for minimal segments

2. **Visual Hierarchy**
   - Name/type at top (primary identification)
   - Objective second (why we're doing this)
   - Training details next (how to do it)
   - Tracking controls last (record progress)
   - Whiteboard hint at bottom (where to learn more)

3. **Color Psychology**
   - Blue: Information, learning, cooperation (partner roles)
   - Orange: Challenge, intensity (resistance)
   - Red: Danger, caution (safety)
   - Green: Success, goals (quality targets)
   - Gray: Secondary, supporting (cues, notes)

4. **Touch-Friendly**
   - Large tap targets for +/- buttons
   - Adequate spacing between elements
   - No tiny text or controls
   - Mobile-first design

5. **Scannable Layout**
   - Clear section headers
   - Consistent icon usage
   - Bullet points for lists
   - Visual boxes for grouped info

## Accessibility

- All icons paired with text labels
- Sufficient color contrast for readability
- Large touch targets (44pt minimum)
- Logical reading order top-to-bottom
- Emoji used for visual interest, not essential meaning

## Responsive Behavior

- Card scrolls vertically on small screens
- Sections stack gracefully
- No horizontal scrolling required
- Text wraps appropriately
- Icons scale with font size

## Performance

- No additional data loading required
- All data mapped from existing models
- Efficient rendering (SwiftUI's built-in optimization)
- Minimal memory overhead (strings and primitives only)

## Interaction Patterns

1. **View Segment Card**
   - Scroll to see all sections
   - Quick visual scan of key info

2. **Track Progress**
   - Tap +/- for rounds
   - Tap +/- for quality metrics
   - Tap checkbox for completion

3. **Reference Information**
   - Read techniques while training
   - Check cues for form reminders
   - Review partner roles mid-session
   - Verify constraints compliance

4. **Seek More Detail**
   - Tap "Whiteboard" toolbar button
   - View comprehensive information
   - See technique breakdowns
   - Review full coaching notes

## Conclusion

The enhanced `SegmentRunCard` provides athletes with essential training information at a glance while maintaining a clean, mobile-friendly interface. The summary-level detail strikes the right balance between "too little" (old version) and "too much" (full whiteboard detail), creating an optimal training tracking experience.
