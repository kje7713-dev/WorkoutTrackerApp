## Color Scheme

### Visual Elements

1. **Group Header (Blue)**
   - Background: Light blue (`Color.blue.opacity(0.1)`)
   - Icon: Blue link icon (`systemName: "link.circle.fill"`)
   - Text: Bold blue text

2. **Exercise Cards (Blue Border)**
   - Border: Light blue stroke (`Color.blue.opacity(0.3)`, 2px)
   - Label: Bold blue text (A1, A2, etc.)
   - Background: System background

3. **Rest Instruction (Orange)**
   - Background: Light orange (`Color.orange.opacity(0.1)`)
   - Icon: Orange pause icon (`systemName: "pause.circle.fill"`)
   - Text: Orange text

4. **Group Container (Gray)**
   - Background: Light gray (`Color(.systemGray6)`)
   - Rounded corners: 16px
   - Padding: 8px

## Key Takeaways

1. **Letter = Group**: All exercises with same letter are a superset
2. **Number = Order**: Execute in numeric order (1, 2, 3...)
3. **Rest After All**: Only rest after completing all exercises in the group
4. **Visual Clarity**: Blue borders and backgrounds indicate grouping
5. **Explicit Instructions**: Text clearly states execution order and rest timing

This implementation eliminates the ambiguity and confusion that caused 90% of superset-related misunderstandings.
