# Exercise Dropdown Feature - Quick Start Guide

## What's New?
The Block Builder now includes a dropdown menu for selecting exercises from a curated library of 50+ standard movements, while still allowing custom exercise names to be entered freely.

## How to Use

### Selecting a Standard Exercise
1. When adding an exercise, tap the dropdown button
2. Browse the list of exercises (filtered by type: Strength or Conditioning)
3. Tap an exercise to select it
4. The exercise name will populate automatically

### Entering a Custom Exercise
**Method 1 - From Dropdown:**
1. Tap the dropdown button
2. Scroll to the bottom
3. Tap "Custom Exercise"
4. Type your custom exercise name in the text field

**Method 2 - Using Toggle Button:**
1. Tap the pencil icon (✎) next to the dropdown
2. A text field will appear
3. Type your custom exercise name
4. Tap the list icon (☰) to switch back to dropdown mode

### Switching Exercise Types
- When you change between Strength and Conditioning, the dropdown automatically updates to show relevant exercises
- Your exercise name is preserved when switching types

## Exercise Library

### Strength Exercises (40+)
The library includes exercises across all major movement patterns:
- Squat variations (Back Squat, Front Squat, etc.)
- Hinge patterns (Deadlift, RDL, etc.)
- Pressing movements (Bench Press, Overhead Press, etc.)
- Pulling movements (Pull-Ups, Rows, etc.)
- Olympic lifts (Clean, Snatch, etc.)
- Core exercises (Plank, Sit-Up, etc.)

### Conditioning Exercises (12+)
Including:
- Monostructural: Assault Bike, Row Erg, Ski Erg, Run, Swimming
- Mixed Modal: Burpees, Box Jumps, Wall Balls, Thrusters

## Benefits
- **Faster**: Select exercises with a single tap
- **Consistent**: Use standardized names across your programs
- **Flexible**: Custom exercises still available when needed
- **Smart**: Automatically filters exercises by type

## Tips
- The dropdown is filtered by exercise type - strength exercises only show for strength, conditioning for conditioning
- Your exercise name is always preserved when toggling between dropdown and text entry
- You can edit a dropdown-selected exercise by switching to custom mode
- All existing blocks with custom exercise names work exactly as before
